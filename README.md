
# 🚢 Titanic Survival Prediction (ML Project)

## 📌 Overview
This project predicts whether a passenger survived the Titanic disaster using Machine Learning.

## 📊 Dataset
- Titanic Dataset
- Features: Age, Gender, Class, Fare, etc.

## ⚙️ Steps Performed
1. Data Cleaning (handled missing values)
2. Data Preprocessing (encoding, scaling)
3. Data Visualization (bar chart, heatmap, etc.)
4. Model Building using Logistic Regression
5. Model Evaluation (Accuracy, Confusion Matrix)

## 🤖 Model Used
- Logistic Regression

## 📈 Results
- Accuracy: ~75% to 85%

## 🚀 How to Run
1. Open Jupyter Notebook
2. Run all cells
3. Check output

## 📌 Conclusion
The model successfully predicts survival and shows how factors like age, gender, and class affect survival.
