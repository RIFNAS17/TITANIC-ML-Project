🚢 Titanic Survival Prediction (ML Project)
📌 Overview
This project predicts whether a passenger survived the Titanic disaster using Machine Learning.

📊 Dataset
Titanic Dataset
Features: Age, Gender, Class, Fare, etc.
⚙️ Steps Performed
Data Cleaning (handled missing values)
Data Preprocessing (encoding, scaling)
Data Visualization (bar chart, heatmap, etc.)
Model Building using Logistic Regression
Model Evaluation (Accuracy, Confusion Matrix)
🤖 Model Used
Logistic Regression
📈 Results
Accuracy: ~75% to 85%
🚀 How to Run
Open Jupyter Notebook
Run all cells
Check output
📌 Conclusion
The model successfully predicts survival and shows how factors like age, gender, and class affect survival.
